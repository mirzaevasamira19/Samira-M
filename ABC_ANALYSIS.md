# ABC-анализ игр магазина цифровых игр

## Цель анализа

Цель ABC-анализа — определить, какие игры приносят основную часть выручки магазина цифровых игр.

## Методика анализа

Для каждой игры была рассчитана выручка по формуле:

Выручка = Цена игры × Количество продаж

Далее была рассчитана общая выручка по всем играм, доля каждой игры в общей выручке и накопительный процент выручки.

После этого игры были разделены на категории:

- Категория A — игры, которые формируют до 80% накопительной выручки;
- Категория B — игры, которые формируют от 80% до 95% накопительной выручки;
- Категория C — игры, которые формируют оставшуюся часть выручки.

## SQL-запрос

ABC-анализ выполнен в файле `abc_analysis.sql`.

В запросе рассчитываются:

- выручка каждой игры;
- общая выручка;
- процент выручки каждой игры;
- накопительный процент выручки;
- категория ABC.

## Вывод

ABC-анализ позволяет определить наиболее прибыльные игры магазина.

Игры категории A являются самыми важными для магазина, так как они формируют основную часть выручки.  
Игры категории B имеют средний вклад в продажи.  
Игры категории C приносят наименьшую часть выручки.

На основе анализа можно принять решение о продвижении игр категории A, а также дополнительно изучить причины низких продаж игр категории C.
